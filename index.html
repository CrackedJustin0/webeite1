Projektarbeit "Webseite"

Diese Seite handelt um die Projektarbeit "Webseite" von Edgar Zinke, Theodor Gebel und Justin Stierwald. Auf diesem Abteil der Seite können sie sich unsere Projektarbeit anschauen, indem sie auf den Knopf unter diesem Text drücken. Auf dem 2. Teil dieser Webseite können sie ein kleines Spiel spielen, welchen sie mit dem obigen Menüpunkt "Spiel" erreichen.
